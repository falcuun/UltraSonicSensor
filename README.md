# UltraSonicSensor
### Wiring:
    - VCC - 5V pin (Breadboard connection, usually the + line on far right or far left).
    - GND - GND pin (Breadboard connection, usually the - line on far right or far left).
    - Trigger Pins:
        - Left:  8
        - Front: 7
        - Right: 4
        - Rear:  2
    - Echo pins: 
        - Left:  9
        - Front: 6
        - Right: 5
        - Rear:  3
